# Evolution

Минимальный прототип на [raylib](https://www.raylib.com/).

- Окно 1560×720: слева мир, справа панель настроек (raygui)
- Ползунки: скорость, радиус, зум, шаг сетки (клик по числу — ввод с клавиатуры)
- Мир 50000×50000 — достаточно для десятков тысяч кружков
- WASD — непрерывное движение

## Сборка

```bash
cmake -B build -S .
cmake --build build
./build/Evolution
```
